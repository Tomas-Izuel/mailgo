## Descripción

Se implementa el microservicio de gestion de stocks basado en las definiciones de la catedra

## Requisitos

- go 1.22.0
- rabbitmq
- mongo
- docker

## Como correr

_Docker_

```
docker-compose up --build
```

_Local_

```
make run
```
